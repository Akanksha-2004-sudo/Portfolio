My Projects

CargoFlow – Freight and Truck Management System

CargoFlow is a monolithic backend project I built to strengthen my skills in Java, Spring Boot, and Hibernate. The application provides complete CRUD operations for managing carriers, trucks, and orders within a single service architecture. I designed and implemented RESTful APIs with clean response structures and maintained strong data integrity through properly validated One-to-Many and Many-to-One entity relationships. The project focuses on writing modular, maintainable code with effective exception handling, resulting in a smoother and more organized order-tracking workflow. This project helped me gain hands-on experience in building scalable backend applications using Spring Boot.

Agribot: An Intelligent Voice Based Chatbot with crop Prediction

I developed Agribot, an intelligent and farmer-friendly chatbot that supports both text and voice interaction to provide agricultural guidance, crop prediction, and real-time soil NPK monitoring. The system integrates natural language processing to enable seamless communication with farmers, while real-time soil parameter analysis assists in accurate crop recommendations and informed decision-making. Designed to enhance agricultural productivity, Agribot combines ease of use with smart data-driven insights, making it a practical digital assistant for modern farming.

Crop Recommendation System

The Crop Recommendation System using Machine Learning is designed to assist farmers in selecting the most suitable crop for cultivation based on real-time soil and environmental data. By analyzing key features such as nitrogen, phosphorus, potassium levels, temperature, humidity, pH, and rainfall, the system uses machine learning algorithms to predict the ideal crop. This not only improves agricultural productivity but also promotes data-driven decision-making in farming practices.
